## Getting Started

### Prerequisites
- Rust 1.75+
- Python 3.11+
- Docker & Docker Compose
- Kubernetes cluster (for production)
- CUDA-capable GPU (recommended for ML training)

### Quick Start

```bash
# Clone the repository
git clone https://github.com/yourusername/ex-machina.git
cd ex-machina

# Run setup script
./scripts/setup.sh

# Start local development environment
docker-compose up -d

# Build all services
./scripts/build-all.sh

# Run tests
./scripts/test-all.sh
```

### Development

Each service can be run independently:

```bash
# Rust services
cd services/sieo
cargo run

# Python services
cd services/sifo
pip install -r requirements.txt
python src/main.py
```
